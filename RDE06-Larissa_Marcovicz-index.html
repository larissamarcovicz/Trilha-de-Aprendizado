Softwares de Versionamento

O que é Git?

O Git é um software livre de versionamento que surgiu para o desenvolvimento do Kernel do Linux, foi projetado e desenvolvido por Linux Torvalds em 2005. Cada diretorio de trabalho do Git é um repositorio com um historico completo e habilidade total de acompanhamento das revisoes, nao dependente de acesso a uma rede ou a um servidor central.

Alguns dos principais comandos do Git

  • commit - Torna permanente um conjunto de mudanças em um repositório.
  • push - Envia mudanças para servidor no repositório remoto.
  • fetch - Verifica se seu repositório local esta em sincronia com ou remoto.
  • pull - Sincroniza o repositório local com o remoto atualizando como versões.

O que é GitHub?

O Github é uma plataforma de hospedagem de codigo-fonte e arquivos com controle de versao. Surgiu em 2008 e foi adquirido em 2018 pela Microsoft por US$7,5 bilhoes, e tambem é um repositório remoto, criado como um serviço online de hospedagem de repositórios do Git.
Link: GitHub

Vantagens em usar o Git e o GitHub

  • Trabalho em equipe - Pode trabalhar com várias pessoas ao mesmo tempo em um mesmo repositório sem problemas.
  • Segurança - Se estragar seu computador você não perde como alterações que fez no repositório já que uma cópia está no servidor remoto do GitHub.
  • Gestão de mudanças - Se você fizer mudanças desastrosas no seu código, pode 'voltar no tempo' para algum commit anterior onde seu projeto estava funcionando corretamente.
